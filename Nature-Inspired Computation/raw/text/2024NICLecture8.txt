Extensions to the Ant Colony 
Optimisation Algorithm
• The original system by Dorigo is known as the Basic Ant 
System (AS) (1996)
• There are other variants that have been found to perform 
better in some problems:
– Max-Min Ant System (Stutzle & Hoos, 2000)
– Elitist Rank Ant System (Bullenheimer, 1999)
• These systems update the pheromone trails differently – using 
only the best or the best n ants to update the trails

ACO is naturally applicable to any sequencing 
problem, or indeed any problem

to the problem as paths in a network 
(construction graph).

network, where, in this case, each job is a 
node. Also, no need to return to start node –
path is complete when every node is visited.

Conclusions
• Ant colony main components:
– Transition Rule
– Update Rule
• Discrete optimisation problems (other than TSP) 
can be optimised using a construction graph
• Results are often competitive with evolutionary 
algorithms
