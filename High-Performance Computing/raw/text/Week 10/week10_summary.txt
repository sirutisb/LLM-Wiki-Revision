Accelerators and GPUs

--- page 2 ---
• GPUs offer good floating point performance, memory bandwidth and energy efficiency
• A significant fraction of the current Top500 systems use GPUs and GPUs also feature 
in forthcoming exa-scale systems
• GPU architectures favour data parallelism where different threads execute the same 
instructions on different data
• The GPU and CPU have separate memory and we need to pay attention to data 
transfers to get good performance
