• Motivation of and introduction to high-performance computing
• Parallel computer architectures: shared-memory and distributed-memory architectures, 
multi-core processors, Graphics Processing Units (GPUs)
• Parallel programming methods: OpenMP, Message Passing Interface (MPI) 
• Floating point arithmetic: floating point model, range, accuracy, exceptions 
• Parallel processing algorithm and programming design: domain decomposition, halo 
exchange, manager-worker, task-based parallelism 
• Parallel performance: speed-up, efficiency, parallel overheads and scaling 
• Interconnection networks in high performance computers: topologies, latency and 
bandwidth
