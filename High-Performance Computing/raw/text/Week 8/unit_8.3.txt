Work sharing constructs
34
• OpenMP provides a number of work sharing constructs which describe how to 
distribute work in a parallel region
• Previously we saw how to use parallel for to parallelise loops
• Two other work sharing constructs are: 
- sections
- single

Single construct
37
• In parallel region work which is not parallelised is repeated by all threads
• The single construct ensures only one thread executes a block of code inside a 
parallel region but it can be any thread
• The single construct was used in the Mandelbrot program in the workshop last 
week 
#pragma omp single [(clause1, ...)]
