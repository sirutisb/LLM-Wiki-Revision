• Modern processor cores can perform multiple floating point operations in a single 
cycle: 
- “Vector” instructions which operate on several numbers at once (e.g. x86_64 AVX instructions)
- Fused multiply-add (FMA: multiply and add in one instruction)
• The number of floating point operations per cycle depends on: 
- The instruction set (vector size, has FMA?)
- Precision (single or double)
• Zen’s Xeon 5560 processors supported the SVE4.2 instruction set which delivered a 
maximum of 4 double precision floating point operations per cycle
