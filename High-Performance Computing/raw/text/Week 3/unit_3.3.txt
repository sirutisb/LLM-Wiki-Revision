Unit Summary
36
‚ÄĘ The advection equation is a partial differential equation containing a time derivative and 
one or more spatial derivatives
‚ÄĘ We can use time stepping to calculate a solution starting from initial conditions like the 
radioactive decay example
‚ÄĘ The spatial derivative can be calculated using a finite difference approximation
‚ÄĘ A boundary condition needs to be specified at the edge of the domain
