Introduction to OpenMP

--- page 2 ---
• OpenMP® (Open Multi-Processing) is a specification for shared memory parallel 
programming with C, C++ or Fortran 
• OpenMP is an open standard (see https://www.openmp.org) 
• The first standard was version 1.0 for Fortran in October 1997 
• C/C++ version 1.0 was released in October 1998 
• The standard continues to evolve and add new features (e.g. support for GPUs, task-
based parallelism) 
• Version 6.0 is the most recent and was released in November 2024 
What is OpenMP?

--- page 3 ---
• OpenMP provides extensions to C, C++ and Fortran
• These extensions enable the programmer to specify where parallelism should be added 
and how to add it
• The extensions provided by OpenMP are: 
- Compiler directives
- Environment variables
- Runtime library routines

--- page 4 ---
• OpenMP uses a fork-join execution model:
- Execution starts with a single thread (master thread) 
- Worker threads start (fork) on entry to a parallel region 
- Worker threads exit (join) at the end of the parallel region
