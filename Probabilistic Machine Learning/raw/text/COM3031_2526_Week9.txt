What Makes RL Different?
Reinforcement learning studies agents that learn by interacting 
with an environment.
• The agent learns through trial and error
• It receives rewards instead of labelled answers.
Key Characteristics of RL
• No supervisor: the agent does not see the correct action, only  
a reward signal.
• Delayed feedback: actions may inﬂuence rewards much later 
in time.
• Sequential decisions: The problem is not i.i.d. , past actions 
inﬂuence future states.
• Actions change the data: The agent’s behaviour affects the 
observations it receives.

Exploration vs Exploitation
Key questions: How should the agent decide which action to take?
• A reinforcement learning agent must balance two strategies:
Exploitation 
• Choose the action that currently seems best and use the knowledge already learned
• Goal: maximise immediate reward
• Example: A robot vacuum follows the path it already knows and cleans efficiently.
Exploration 
• Try new actions to gather more information
• Discover potentially better strategies
• Example: the robot vacuum tries a new route it has never used before.
Key Challenge
• Only exploiting →may miss better actions
• Only exploring →inefficient learning
Good RL balances exploration and exploitation

• Exploit: Go to your favourite restaurant 
• Explore: Try a new restaurant that might 
be better 
Online Banner Advertisements 
• Exploit: Show the most successful advert 
• Explore: Show a different advert
