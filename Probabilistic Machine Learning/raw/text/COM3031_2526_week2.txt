Summary
17
Linear regression: 
• Assumes Gaussian noise, constant variance, and continuous outputs
• Not suitable for binary or count data
Generalised Linear Models (GLMs):
• Extend linear models to non-Gaussian data
• Still use a linear predictor
Key differences from linear regression:
• The response y can follow non-Gaussian distributions (e.g. Poisson, Bernoulli)
• Errors do not need to be normally distributed
• Variance can depend on the mean
Link function connects the mean of the distribution to the linear predictor and 
Can be non-linear (e.g. log link in Poisson regression)
