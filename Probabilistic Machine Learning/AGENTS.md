# Agent Instructions

This repository uses the shared wiki conventions in `CLAUDE.md` and `GEMINI.md`.

For Codex and other coding agents:

- Read `CLAUDE.md` or `GEMINI.md` before editing wiki content.
- Treat `raw/` as immutable source material.
- Put generated revision material under `wiki/`, using lowercase hyphenated filenames.
- Use Obsidian-style wikilinks: `[[filename-without-extension]]`.
- Update `wiki/index.md` when adding a broadly useful page.
- Append a dated entry to `wiki/log.md` after creating or materially changing wiki content.
- Preserve the module's formula-status distinction: Weeks 3-5 and 7-9 require memorised formulas; Week 9 has no formula sheet and focuses on multi-armed bandits and Q-learning.
